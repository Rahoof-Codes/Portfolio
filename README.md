## ✨ About

A clean, responsive personal portfolio built with **React (CDN + Babel)** — no build tools, no npm — designed to run directly in **SPCK Editor** on mobile.

---

## 🛠️ Tech Stack

| Layer | Tech |
|-------|------|
| Structure | HTML5 |
| Styling | CSS3 (custom, no framework) |
| Logic | React 18 (via CDN) |
| JSX | Babel Standalone |
| Fonts | Google Fonts (Poppins + Space Mono) |
| Hosting | GitHub Pages / Vercel |

---

## 📁 File Structure

```
Portfolio/
├── index.html      # App shell + all CSS styles
├── config.js       # ✏️ All personal data (edit this to update portfolio)
├── app.js          # React components
├── images/
│   ├── rah.jpg     # Profile photo
│   ├── HTML.jpg    # Certificate
│   └── CSS.jpg     # Certificate
└── README.md
```

> **To update content** — only edit `config.js`. No need to touch `app.js`.
